# PhotoAlbum_Postrero

A simple ASP.NET Core MVC photo album web application featuring curated scene galleries from popular anime movies.

## Prerequisites

- [Visual Studio 2022](https://visualstudio.microsoft.com/) (or later) with the **ASP.NET and web development** workload installed
- [.NET 10.0 SDK](https://dotnet.microsoft.com/download)

## How to Run (Visual Studio)

1. Open `PhotoAlbum_Postrero.slnx` in Visual Studio
2. Press **F5** (or click the green â–¶ **Start** button) to build and run
3. The app will open in your browser automatically

## How to Run (Command Line)

```bash
cd PhotoAlbum_Postrero
dotnet run
```

Then open the URL shown in the terminal (e.g. `https://localhost:5001`) in your browser.
